# Log Hari 1 — Dragon Ball Gacha

Proyek Hari 1 telah dimigrasikan dari Poké Gacha ke **Dragon Ball Gacha**.

Versi saat ini menggunakan Dragon Ball API untuk mengambil data karakter asli,
kemudian menampilkannya sebagai hasil summon. Detail implementasi, fitur, dan
verifikasi dicatat pada [dragonball.md](dragonball.md).
